## O que é um algoritmo?

Um algoritmo é uma receita ou procedimento bem definido que transforma uma entrada (input) em uma saída (output) em tempo finito. Por exemplo, uma lista de números [26,31,41,41,58,59] é uma instância do problema de ordenação; o algoritmo deve produzir uma permutação ordenada dessa lista.

De maneira informal, um algoritmo é qualquer procedimento computacional bem definido que recebe algum valor, ou conjunto de valores, com entrada e produz algum valor, ou conjunto de valores, com saída em um tempo finito. É uma seqüência de passos computacionais que transformam a entrada na saída.

## What is an algorithm?

An algorithm is a well-defined recipe or procedure that transforms an input into an output in finite time. For example, a list of numbers [26,31,41,41,58,59] is an instance of the sorting problem; the algorithm must produce a sorted permutation of this list.

Informally, an algorithm is any well-defined computational procedure that takes some value, or set of values, as input and produces some value, or set of values, as output in finite time. It is a sequence of computational steps that transforms the input into the output.
